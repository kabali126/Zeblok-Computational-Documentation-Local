  <Accordion title="🚀 v2.0.0 — May 2026 (Major Release)">
    **Our biggest release yet.** A complete platform evolution — rebuilt from the ground up. This release introduces AI-Adaptive Cache acceleration and strengthens authentication and security across the platform.

    **🌟 Top things in v2.0.0**

    - 🚀 Brand-new AI-MicroCloud® platform experience
    - ⚡ AI-Adaptive Cache acceleration
    - 🔐 Stronger security across every layer
    - 🤖 Smarter agents with OpenClaw
    - 💰 TCO Calculator — available exclusively as part of Zeblok Labs
    - 🩺 Health Check Application — built-in diagnostics across your environment

    **SaaS Layer**

    - **AI-Adaptive Cache Plans** — A new dedicated plan category that lets you launch workloads on AI-Adaptive Cache hardware, with the right configuration applied automatically.
    - **AI-Adaptive Workstations** — Spin up workstations with cache acceleration to dramatically speed up model loading and data-heavy workflows.
    - **Memory Offloading for Fine-Tuning** — Fine-tune larger models on smaller GPU footprints by offloading memory dynamically — train models that previously required premium hardware.
    - **Inference Offloading with KV Cache Reuse** — Long chat sessions and multi-turn conversations now run faster and cheaper, thanks to intelligent KV cache reuse across requests.
    - **Redesigned Datacenter Page** — A complete view of every datacenter's compute, GPU, and memory resources at a glance.
    - **Unified Workstation Configuration** — A single, streamlined screen to configure and launch workstations — fewer clicks, faster setup.
    - **Multi-Tenant Organization Structure** — A cleaner organizational model giving enterprise customers stronger separation between teams, business units, and projects.
    - **Namespace Member Management** — Easily add, edit, and view team members directly from any namespace, making collaboration faster across your organization.
    - **Live Microservice Image Updates** — Update the underlying image of a running microservice without losing state or redeploying from scratch.
    - **Enhanced Cluster Cards** — At-a-glance details on every cluster, including region and key configuration, right from the dashboard.
    - **Unified Private Registry Imports** — A single, consistent way to bring images from your private registry into the platform — whether for workstations or microservices.
    - **TCO Calculator** *(Available exclusively as part of Zeblok Labs)* — Calculate your total cost of ownership and see exactly how much you can save over the next 5 years with memory offloading enabled. The calculator quantifies the long-term GPU savings you'll unlock by running larger models on smaller, more efficient hardware footprints.
    - **Secure Inference Access with GenAI Keys** — A new key-based access layer makes it easy to securely share and govern who can call your inference endpoints.
    - **Token-Based Inference Authentication** — Every inference call is now backed by token authentication, giving you full control and audit visibility over API usage.
    - **Inference Utilization Dashboard** — Track real-time inference performance with built-in metrics like Time to First Token (TTFT) and Time per Output Token (TPOT) — essential for monitoring user experience.
    - **Historical Inference Analytics** — Performance metrics are now persisted, so you can analyze trends, plan capacity, and tune workloads over time.
    - **AI-Adaptive Cache Monitoring** — Watch SSD read/write activity and usage in real time to fine-tune cache-accelerated workloads.
    - **Health Check Application** — A built-in diagnostic app that runs end-to-end health validation across your environment, surfacing issues before they impact production and giving your operations team confidence in cluster readiness at any moment.

    **IaaS Layer**

    - **Edge Platform Generally Available** — Our Edge variant is now production-ready, purpose-built for distributed and on-prem deployments.
    - **GPU Workloads on Edge** — Run accelerated AI workloads directly at the edge, closer to your data.
    - **DDIL Support for the Edge** — Resources running on the edge node remain accessible locally even when the connection between the HUB and the Edge breaks, with local visibility surfaced in the UI.
    - **Wider Hardware Compatibility on Edge** — The platform now runs on edge hardware without AVX support, opening up more deployment options.
    - **Automatic NVMe Disk Setup** — NVMe disks are partitioned and prepared automatically during bare-metal setup, eliminating one of the trickiest manual steps.
    - **Multi-Cluster Domain Management** — Run multiple clusters under dynamically managed domains — ideal for multi-region or multi-environment setups.
    - **Account Security Essentials** — Email verification at signup, license key validation, and full Forgot Password support — all built in.
    - **Standalone AI-Adaptive Cluster Option** — A simpler single-node AI-Adaptive deployment for smaller environments and pilots.
    - **Hardened Kubernetes Control Plane** — Stronger security baselines applied across master and worker nodes for production-grade protection.
    - **Encrypted Cluster State** — All cluster state in ETCD is now encrypted at rest, protecting sensitive configuration and secrets.
    - **Built-In Audit Trails** — Comprehensive auditing and logging for compliance, traceability, and incident response.
    - **Default-Deny Network Isolation** — Workload isolation strengthened with deny-all network policies, reducing lateral attack surface.
    - **Tightened Server Access** — Server access locked down to trusted endpoints only, drastically reducing exposure.
    - **Three-Tier Role-Based Access Control** — New Admin, User, and Superadmin roles give you precise control over who can do what.
    - **AES-256 Encryption at the Backend** — Sensitive data is now encrypted at the backend with AES-256, the industry standard.
    - **Strong Password Policies** — Enforced password strength rules across every user account.

    **PaaS Layer**

    - **Flexible Environment Seeding** — Choose default or fully custom seeding to populate your environment exactly the way you want, with full log visibility throughout.
    - **Pick & Choose What to Seed** — Select exactly which workstations, microservices, agents, and models to seed, with clear status markers showing what's already loaded.
    - **Pre-Configured Admin Identity** — A single admin identity is provisioned across the platform, with secure credentials delivered straight to your inbox.
    - **One-Click Seed Initialization** — Kick off seeding directly from the cluster card, with smart enabling that prevents action before your cluster is ready.
    - **Faster Catalog Browsing** — Smart caching means catalog screens load near-instantly, even with large environments.
    - **Universal Model Downloader** — Pull models from AWS, Azure, or MinIO — the platform automatically detects and uses the right backend.
    - **Validated for Popular LLMs** — Out-of-the-box compatibility tested with Mistral-7B and Llama 3.x (including 70B variants).
    - **Latest MLflow Integration** — Updated to MLflow 3.9.0 with a refreshed model-serving flow.
    - **Refreshed Catalog** — Workstation images and core microservices updated to the latest stable versions.
    - **Introducing OpenClaw** — A new agent runtime that supports both self-hosted LLMs and external APIs, giving you full flexibility on how you run agents.
    - **Memory-Offload Inference** — Run larger models on smaller hardware footprints with our new memory-offloading inference mode.
    - **Smarter Agent Backbone** — Agents now run on Qwen3-Coder-Next FP8 (80B MoE), delivering significantly better instruction-following and reasoning.
    - **Custom Agent Personas** — Define unique personas and skill permissions per agent — perfect for tailored, role-specific assistants.
    - **MCP Server Integration** — Plug external tools and services into your agents using the MCP standard.
    - **API Key Management** — Issue, rotate, and manage API keys for programmatic platform and inference access.

    **Documentation**

    - **Brand-New Documentation Portal** — A redesigned public docs experience launched at https://zeblokcomputationalprivatelimited.mintlify.app/

    **Bug Fixes**

    - Resolved errors when creating namespaces and inviting or creating users.
    - Fixed inconsistent behavior when selecting models from dropdowns.
    - NVMe disk listings now show only relevant, valid disks — cleaner and easier to navigate.
    - Fixed an issue that prevented saving updated workstation images during onboarding.
    - Resolved image version inconsistencies when launching workstations.
    - Fixed the "Selected Plan is not available" error when launching workstations with custom plans.
    - Workstation and microservice launches now correctly apply your specified CPU and memory limits.
    - Volume paths are now created automatically — no more manual entry during deployment.
    - Stopped or deleted inferences no longer clutter the agent setup list.
    - Fixed Azure cluster name display issues and the Azure log download flow.
    - Storage buckets and model-hub data are now reliably attached to launched environments.
    - Downloaded models are now stored in the proper format for full compatibility.
    - Adding worker nodes to bare-metal clusters now works reliably end-to-end.
    - Default seed data is preserved across service restarts.
  </Accordion>

    - **Security Scan Log Refresh:** Improved refresh performance for security scan logs.
